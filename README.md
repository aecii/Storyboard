Storyboard

A browser-based visual board for arranging images, notes, and drawings on a freeform canvas.

Features
- Drag & drop images (move, resize, layer)
- Sticky notes (editable, draggable)
- Freehand drawing + eraser
- Undo (Ctrl+Z)
- Auto-save (restores on reload)
- Export board as PNG

Storage
IndexedDB → images
localStorage → layout, notes, drawings

Usage
Add images → drag & drop or upload
Move/resize → drag header / corner
Add note → “+ Note”
Draw → toggle “Draw”
Export → “Export PNG”


Run
open storyboard.html

Notes
No redo (undo only)
Desktop-focused
Data is stored locally (no cloud sync)
